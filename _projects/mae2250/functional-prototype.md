# Functional Prototype

## Overview

The functional prototype demonstrates a **mechanical trapping system** designed to capture and contain spotted lantern flies (SLFs) during vineyard harvest. The system integrates four main subsystems: container, drawer, funnel mechanism, and extendable pole.

## Subsystem Breakdown

### Container
- **Function:** Outer housing for funnel and drawer  
- **Material:** Wood (Taylor Design Studio)  
- **Fabrication:** Cut using bandsaw and assembled using adhesive  

---

### Drawer
- **Function:** Stores captured SLFs and liquid attractant  
- **Material:** Wood + CA glue  
- **Fabrication:**  
  - Built box structure  
  - Added internal rails for sliding  
  - Designed for easy insertion/removal  

---

### Funnel Mechanism
![Funnel Sketch](/assets/images/slf-funnel-sketch.png)

- **Function:** One-way entry system preventing escape  
- **Materials:** Wire ring, wood flaps, rubber bands/string  
- **Fabrication:**  
  - Cut wooden flaps using bandsaw  
  - Added notches for string/rubber band  
  - Drilled holes for mounting  
  - Bent wire into ring and attached flaps using zip ties  

---

### Extendable Pole
![Pole Sketch](/assets/images/slf-pole-sketch.png)

- **Function:** Adjustable mounting system for height positioning  
- **Material:** Wood + cylindrical rods  
- **Fabrication:**  
  - Created telescoping horseshoe-shaped structure  
  - Drilled holes every 2 inches  
  - Used pin system for height locking

## Testing Plan

### Container
- **Goal:** Structural stability under wind load (~10 mph)  
- **Method:** Apply horizontal force using rope + scale  
- **Success Criteria:** No rotation, deformation, or displacement  

---

### Drawer
- Load capacity: 12 kg  
- Smooth operation without jamming  
- Withstands 50 open/close cycles  

---

### Funnel
- Flaps meet within 1–3 cm  
- Allows entry but prevents escape  
- Adjustable tension using rubber band/string  

---

### Extendable Pole
- Adjustable without slipping  
- Withstands repeated use (50 cycles)  
- Supports full system load (~14.25 kg)  

---

## Testing Results

### Drawer Performance
- Successfully held **6 kg load** (partial test case)  
- Smooth motion over **15 cycles without jamming**  
- No visible damage after repeated use  
- **Limitation:** Handle design needs improvement  

---

### Funnel Performance
- Under proper configuration:  
  - SLF model enters easily  
  - Cannot escape once inside  

- Failure cases observed:  
  - Improper flap spacing  
  - Twisted flaps  
  - String interference  

---

### Extendable Pole
- Successfully adjusted **50 times without damage**  
- Maintained stability under load  
- No visible deformation  

---

## Key Design Insights

- Funnel performance is highly sensitive to flap alignment  
- Drawer mechanism is reliable but ergonomics need refinement  
- Pole system is structurally sound and requires minimal redesign  

---

## Design Improvements (Next Iteration)

- Add spacers to maintain flap alignment  
- Introduce multiple attachment points for flaps  
- Limit flap overextension using secondary ring  
- Make flap length adjustable  
- Reduce flap thickness to prevent interference  
- Explore **octagonal funnel geometry** for better sealing  

---

## Success Criteria

The prototype is considered successful if:

- Drawer supports **12 kg load** without failure  
- Drawer operates smoothly for **50 cycles**  
- Pole adjusts reliably without slipping  
- Funnel allows entry but prevents escape consistently  
- Flaps maintain controlled range of motion
